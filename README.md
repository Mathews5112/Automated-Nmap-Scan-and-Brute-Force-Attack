# Automated Nmap Scan and Brute Force Attack

This project performs automated network scanning and brute force attacks using Nmap.

## Demo

Watch the project in action:

[![Demo Video](Brute_Force.mp4)](Brute_Force.mp4)

Or click the link above to view the video directly.

## Features

- Automated network scanning with Nmap
- Brute force attack capabilities
- Network vulnerability assessment

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python "nmap scan and brute force.py"
```
